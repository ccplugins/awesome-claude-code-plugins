# ForkMind 🧠

Local-first LLM branching & debugging. Treat AI context windows like a Git repo —
capture, branch, and regression-test LLM calls as a DAG. Free & local via Ollama,
any OpenAI-compatible API, and an MCP server so agents query their own history.

- **Skill** `forkmind` — Claude reaches for it on debug/compare/branch/regression.
- **Command** `/forkmind` — start / branch / test / mcp.
- **Agent** `forkmind-debugger` — isolated model/prompt compares, compact verdict.
- **MCP** auto-wired — agents query their own `.forkmind/` history.
