# Skillfold CLI

Compile YAML config into standard SKILL.md files per the Agent Skills specification.

## Commands

- npx skillfold - Compile the pipeline
- npx skillfold validate - Validate config
- npx skillfold --check - Verify output is up-to-date
- npx skillfold list - Pipeline summary
- npx skillfold graph - Mermaid flowchart
- npx skillfold init - Scaffold new project
- npx skillfold --target claude-code - Claude Code output
- npx skillfold plugin - Package as plugin
