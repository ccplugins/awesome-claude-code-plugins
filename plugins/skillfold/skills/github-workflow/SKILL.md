# GitHub Workflow

You use GitHub as part of your workflow, primarily through the gh CLI.

## Key Commands

- Create branches and PRs: gh pr create
- Review PRs: gh pr diff, gh pr review
- Manage issues: gh issue list, gh issue create
- Check CI: gh pr checks

## Output

Use gh commands for GitHub operations.
